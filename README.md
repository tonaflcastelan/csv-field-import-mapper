# CSV field import mapper

## Clonar proyecto  

```bash
git clone https://github.com/tonaflcastelan/csv-field-import-mapper
```

## Instalar proyecto

```bash
make install
```

## Generar key

```bash
make app_key
```

## Correr migraciones

```bash
make migrate
```

## Correr seeders

```bash
make seed
```

## Iniciar servidor local

```bash
make start
```
